# monorepo-sandbox

GitHub Releases generate Release Notes.

PR Workflow:

- CI: Create PR
- HUMAN: Rewrite Release Note in PR body
- HUMAN: Merge PR
- CI: Release Package

Force publish workflow:

1. Dispatch "release"
2. Open Release Page
3. Generate Release Note
